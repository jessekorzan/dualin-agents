🤖 What is Duelin' Agents?

A revolutionary dual-agent chat interface that lets AI agents have conversations with each other in real-time. Watch as different AI personalities debate, collaborate, and create together!

  • Split-screen chat showing both agents
  • Real-time streaming responses
  • Turn-based conversation flow
  • Visual status indicators

⚡ Perfect with n8n

Duelin' Agents integrates seamlessly with n8n workflows, making it incredibly easy to connect any AI service via webhooks.

  • Connect OpenAI, Claude, Gemini, or any AI API
  • Custom variable names for your workflows
  • Session management for conversation context
  • Error handling and rate limiting built-in
